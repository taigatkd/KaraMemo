# 設定・テーマ・広告設計

## 1. カラオケ機器設定

### 対象機器

- `DAM`
- `JOYSOUND`

### 設定項目

- `BGM`
- `マイク`
- `エコー`
- `Music`

### 値域

- `0..50`
- 初期値はすべて `25`

### UI

- 曲一覧画面の AppBar から開く
- ハーフモーダル表示
- 機器切替はチップ UI
- 値変更はスライダー UI

## 2. テーマ

- ライトテーマとダークテーマを持つ
- OS 設定に追従する
- 主要コンポーネントの色はテーマから取得する

## 3. 広告

### 表示方針

- 主要画面下部にバナー広告を表示する
- 複数選択モードでは広告非表示

### 読み込み状態

- 読み込み成功時は広告表示
- 読み込み中はプレースホルダー表示
- 読み込み失敗時もプレースホルダー表示

### セキュリティ方針

- 広告IDはコードへ直書きしない
- `local.properties`、Gradle 変数、または安全な設定注入を使う
- テストモード切替を用意する

## 4. Android 側の保存方針

- 現在のカラオケ機器は `DataStore` 保存
- 機器ごとの設定値も `DataStore` 保存
- テーマはシステム追従を初期方針とする
