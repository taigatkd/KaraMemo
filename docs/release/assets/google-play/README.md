## Notes

- Screenshots were captured from the seeded debug build on the Android emulator.
- `ja-JP/` contains the raw captures from the emulator.
- `ja-JP/play-ready/` contains the 9:16 portrait versions intended for Play Console upload.
- `ja-JP/store-ready/` contains the more stylized Play Store listing images with headings and short explanatory copy.
- The feature graphic is based on the real app screenshots and launcher icon.
